## Adding Your App

To add a new application:

1. Create a new directory with your app name
2. Add your application files
3. Include a README.md with:
   - App description
   - Installation instructions
   - Usage examples
   - Dependencies

See [CONTRIBUTING.md](../CONTRIBUTING.md) for detailed guidelines.

## App Organization

Each app should be self-contained in its own directory:

```
apps/
├── app-1/
│   ├── README.md
│   ├── src/
│   └── ...
├── app-2/
│   ├── README.md
│   ├── src/
│   └── ...
└── ...
```
