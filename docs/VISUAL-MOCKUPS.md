## Responsive Behavior

### Desktop (1366x768)
- Full sidebar navigation (200px)
- Multi-column layouts
- All features visible

### Tablet (1024x768)
- Condensed sidebar (60px, icons only)
- Two-column max
- Touch-optimized targets

### Mobile (Portrait)
- Hidden sidebar (hamburger menu)
- Single column
- Essential features only
- Reference mode

## Animation Notes

All transitions use **ease-out** timing:
- Fast: 150ms (hover effects)
- Standard: 250ms (page transitions)
- Slow: 400ms (modals)

Animations are subtle and professional, matching the Star Trek LCARS aesthetic.

## Accessibility Features

- **Keyboard Navigation**: Full tab support
- **Screen Reader**: Proper ARIA labels
- **High Contrast**: Option available
- **Font Scaling**: Respects system settings
- **Focus Indicators**: Clear blue outline
